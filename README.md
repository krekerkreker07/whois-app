## Run WhoisApp

### Add execution rule to sh file
- `chmod +x build.sh run.sh`

### Build app 
- `./build.sh`

## Run app
- `./run.sh`

### Web: 
http://localhost:8080/

### Api: 
http://localhost:8080/api/whois?domain=en.wikipedia.org
http://localhost:8080/api/whois?domain=trust-view.com.ua

## Stop app
- `./stop.sh`
